동글랜드 마인크래프트 서버용 Fabric 모드 자동 설치 런처

## 설치
[Releases](https://github.com/Garamisme/dongleland_Mod_Launcher/releases)에서 최신 exe 다운로드

- 셰이더팩 관리
- Java/Fabric 자동 설치/업데이트
